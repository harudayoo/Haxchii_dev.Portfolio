# Harold Daniel Cayan | Portfolio

A modern, responsive portfolio website showcasing my work as a Full Stack Web Developer. Built with HTML5, CSS3, and JavaScript, featuring smooth animations and professional design.

## About Me

I'm Harold Daniel S. Cayan, a Full Stack Web Developer and IT Hardware & Software Specialist based in Davao City, Philippines. I hold a BS in Information Technology, Major in Information Security from the University of Southeastern Philippines (Cum Laude).

## Features

- **Responsive Design** - Optimized for all devices and screen sizes
- **Smooth Animations** - AOS (Animate On Scroll) library for engaging UX
- **Professional UI/UX** - Clean design with gradient backgrounds and modern effects
- **Performance Optimized** - Fast loading with efficient animations
- **Cross-browser Compatible** - Works on all modern browsers
